# PokéCards Market

Aplicación web interactiva para explorar y comprar cartas coleccionables utilizando una API pública y pagos con PayPal Sandbox. El proyecto se entrega en formato `.zip`.

## Tecnologías utilizadas

* React / Vite
* JavaScript
* CSS
* PokéAPI
* PayPal Sandbox

---

# Requisitos

Antes de ejecutar el proyecto asegúrese de tener instalado:

* Node.js v18 o superior
* npm
* Git (opcional)

---

# Instalación del proyecto

## 1. Descomprimir el archivo ZIP del proyecto

Extraer el contenido del archivo `.zip` en cualquier carpeta del equipo.

## 2. Entrar al proyecto

```bash
cd Parcial-2-Componentes-Web
code .

```

## 3. Instalar dependencias

```bash
npm install
```

---

# Ejecutar el proyecto

## Modo desarrollo

```bash
npm run dev
```

Luego abrir en el navegador:

```txt
http://localhost:5173
```
